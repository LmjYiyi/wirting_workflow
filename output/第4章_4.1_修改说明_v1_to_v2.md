### 修改1:优化引言段的因果逻辑(优先级1)

**位置**: 第1段(章节引言)

**原文问题**:
- 第一句话信息密度过高,存在"概念堆砌"感
- 因果逻辑不够明确

**修改前**:
```
研究表明:当判据 $B \cdot \eta \cdot \tau_0 > 1$ 时,色散引入的频率展宽 $\Delta f_D$ 远超FFT频率分辨率 $\delta f = 1/T_m$,频谱主瓣严重散焦,传统峰值检测方法已不可避免地失效。在这种强色散条件下,差频信号从单频稳态信号退化为时变调频信号,全频段FFT分析面临根本性的逻辑矛盾——频率尺度的非线性扭曲导致频域映射关系失效,无法从模糊且畸变的频谱中直接提取物理参数。
```

**修改后**:
```
研究表明,当判据 $B \cdot \eta \cdot \tau_0 > 1$ 时,色散引入的频率展宽 $\Delta f_D$ 远超FFT频率分辨率 $\delta f = 1/T_m$,频谱主瓣严重散焦,传统峰值检测方法必然失效。这种失效的根本原因在于:强色散条件下,差频信号从单频稳态信号退化为时变调频信号,其频率尺度发生非线性扭曲,导致全频段FFT分析面临无法自洽的逻辑矛盾。具体而言,频域映射关系本身依赖于待反演的色散参数,而参数反演又需要预知频域映射,形成了"已知量与未知量相互依赖"的循环逻辑,使得从模糊且畸变的频谱中直接提取物理参数成为不可能。
```

**改进要点**:
1. 拆分长句,在"失效"后断句
2. 用"这种失效的根本原因在于"明确指向因果关系
3. 深化"循环逻辑"的阐述,增强可理解性
4. 术语优化:"已不可避免地失效" → "必然失效"

### 修改2:补充符号P与$\hat{P}$的关系说明(优先级5)

**位置**: 式(4-8)附近(4.1.2节)

**原文问题**:
- 式(4-8)中使用了$P$表示路径数,但未明确说明这与后续的$\hat{P}$的关系

**修改前**:
```
假设在第 $i$ 个滑动窗口内,接收到的差频信号包含 $P$ 个路径分量(直达波+多径反射波)。后文将通过MDL准则估计该路径数,记为 $\hat{P}$。信号模型可表示为:

$$x(n) = \sum_{p=1}^{P} A_p e^{j(2\pi f_p n + \phi_p)} + w(n), \quad n = 1, 2, \ldots, M \tag{4-8}$$

其中 $f_p$ 为第 $p$ 个路径的归一化差频,...
```

**修改后**:
```
假设在第 $i$ 个滑动窗口内,接收到的差频信号包含 $P$ 个路径分量(直达波+多径反射波)。信号模型可表示为:

$$x(n) = \sum_{p=1}^{P} A_p e^{j(2\pi f_p n + \phi_p)} + w(n), \quad n = 1, 2, \ldots, M \tag{4-8}$$

其中 $P$ 为真实信源数(未知量),后文将通过MDL准则估计该路径数,记为 $\hat{P}$。

其中 $f_p$ 为第 $p$ 个路径的归一化差频,...
```

**改进要点**:
1. 将$P$的定义移至式(4-8)之后,紧跟公式
2. 明确标注$P$为"未知量",$\hat{P}$为"估计值"
3. 增强符号定义的严谨性
